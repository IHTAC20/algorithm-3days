# algorithm-3days

1주일에 3일 이상 알고리즘 푸는 스터디

### RULE

- repo를 자신의 깃으로 fork 한다.
- 일주일에 세 번, 플랫폼에 상관없이 알고리즘 문제를 푼다.
   - 풀다가 마주친 오류, 실수를 포함한 풀이를 기록(주석, 리드미 등등)
- 일요일까지 PR을 올린다.
   - **PR을 올리지 않으면 벌칙금 1,000원(일당)**
- 100문제 풀때까지 스터디는 계속된다:)
